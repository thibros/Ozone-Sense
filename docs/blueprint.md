# **App Name**: OzoneSense

## Core Features:

- Parameter Input Form: Users can input space volume (cubic meters), select ozonator strength (15000/30000 mg/h), set ozone half-life, and define the safe concentration limit.
- Ozonator Schedule Builder: An interactive interface for users to add a sequence of 'on' and 'off' states for the ozonator, up to 180 minutes.
- Ozone Concentration Calculator: Core logic to simulate ozone generation, decay, and calculate concentration levels over time based on user inputs.
- Tabular Output Display: A table showing calculated ozone concentration (mg/m³) at 30-minute intervals until it falls below the safe limit.
- Concentration Trend Graph: A dynamic chart visually representing the ozone concentration over time, mirroring the tabular data.

## Style Guidelines:

- Primary color: A clear and analytic medium-dark blue (#2E78C4), conveying trust and precision.
- Background color: A very light, desaturated blue-gray (#F3F6FA), providing a clean and calm canvas.
- Accent color: A vibrant, analogous cyan (#25C9E5) for highlights and interactive elements, symbolizing air and environment.
- Headline and body font: 'Inter' (sans-serif) for its modern, objective, and highly legible characteristics, suitable for data-driven applications.
- Use clean, outlined, geometric icons that visually reinforce measurement, settings, and data representation.
- A single-page, responsive layout clearly separating input controls from data visualization, emphasizing spaciousness and ease of navigation.
- Subtle, fluid transitions and micro-animations when updating charts or displaying results to enhance user experience without distraction.
